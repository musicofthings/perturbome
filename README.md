# Perturbome Atlas

A static, responsive educational website about perturbation biology, with Nanopore long-read perturbomics presented first and in scientifically bounded terms.

## Open locally

Open `index.html` directly, or run:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy on GitHub Pages

1. Create a GitHub repository and copy the contents of this folder to the repository root.
2. Commit and push to the `main` branch.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/ (root)`, then save.

No build process or package installation is required.

## Scientific positioning

The site describes Nanopore as the strongest choice when the primary information need is full-length transcript structure, isoform switching, direct edit evidence, or native-RNA features. It does not claim that Nanopore has the lowest cost or highest screen scale. The recommended architecture is multimodal: scalable discovery followed by long-read mechanistic resolution.
